Простая работа с данными в байтовом представлении.

Библиотка javascript/typescript (ES6) для node.js.

- преобразование строки в байты,
- конвертация между двоичным, строковым, десятичным и шестнадцатеричным представлениями,
- работа с битовыми масками,
- операции and, or, xor,
- побитовый сдвиг.

Библиотека написана для простой и быстрой работы. Все эти операции есть в виде нативных методов. Но мы постарались сделать наши методы наиболее быстрыми и удобными.

# Установка

```
npm install lib-bytes
```

или

```
yarn add lib-bytes
```

# Начало работы

Можно импортировать методы как отдельные функции:

```
import { hex2bin } from 'lib-bytes';

hex2bin('d2');
```

или вызывать их как статические методы объекта:

```
import * as bytes from 'lib-bytes';

bytes.hex2bin('d2');
```
